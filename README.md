# KWTConverter
Converts a TestComplete KeyWordTest file from XML to text

Supported features:
- Keywords:
  - Run KeywordTest
  - Run external library function
  - Run Script
  - Catch
  - Comment
  - Data-Driven Loop
  - If/Else
  - For Loop
  - Group
  - Log.Error
  - Return
  - Set Variable Value
  - Try
